Release

  • Update and merge the Pull Request with label autorelease:pending managed by Release Please:

    • checkout release-vX.Y.Z branch

    • set version to X.Y.Z in docs/antora.yml

    • set target_revision 's default to vX.Y.Z in modules/variables.tf

    • set version in [params] section to X.Y.Z in site/config.toml

    • amend the commit

    • push force the branch

    • merge the PR
