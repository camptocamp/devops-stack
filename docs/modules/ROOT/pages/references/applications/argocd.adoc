= ArgoCD

This chart installs and configures:

* ArgoCD itself
* A Grafana dashaboard for ArgoCD
* An `OAuthClient` Kubernetes object if the OpenShift 4 option is
enabled

== Requirements

[cols=",,",options="header",]
|=====================================================
|Repository |Name |Version
|https://argoproj.github.io/argo-helm |argo-cd |2.14.7
|=====================================================
