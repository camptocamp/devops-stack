== Clone the project 

[source,shell,subs="attributes"]
----
$ git clone {url-repo}
$ cd {project-name}
----

== Run the stack

=== Create a cluster

Just run :
[source,shell]
----
$ make
----

It will:

- spawn a K3s cluster on your workstation using Docker
- deploy ArgoCD in the cluster
- deploy the App of Apps that manages ArgoCD and itself.

=== Destroy a cluster

Just run :
[source,shell]
----
$ make clean
----
