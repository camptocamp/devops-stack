Run an example

git clone git@github.com:camptocamp/devops-stack.git
cd devops-stack/tests/k3s-docker/terraform

Run the stack

Launch a demo cluster

Just run :

terraform init
terraform apply

It will:

  • spawn a K3s cluster on your workstation using Docker

  • deploy ArgoCD in the cluster

  • deploy the App of Apps that manages ArgoCD and itself.

  • deploy a demo Application

Destroy a cluster

Just run :

terraform destroy
